সরাসরি মোবাইলে ইনস্টল করুন

কোনো প্লে-স্টোর ছাড়াই অ্যাপের মতো ব্যবহার করুন

Android (Chrome / Opera)

  1. ব্রাউজারের ডানদিকের উপরে (৩ ডট) আইকনে চাপুন।
  2. মেনু থেকে Install App অথবা Add to Home Screen-এ চাপুন।
  3. কনফার্ম করলেই ২ সেকেন্ডে অ্যাপটি ফোনে ইনস্টল হয়ে যাবে!

iPhone / iOS (Safari)

  1. সাফারি ব্রাউজারের নিচে থাকা Share (শেয়ার) বাটনে চাপুন।
  2. নিচের অপশনগুলো থেকে স্ক্রল করে Add to Home Screen চাপুন।
  3. ডানদিকের উপরে Add (যোগ করুন) বাটনে ক্লিক করুন।
